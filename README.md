# antPlayer
An android music player based on BASS Audio Library. This application can play music with lower latency than the default Android music player. File types supported by this application is also much more.

# Supported File Type
|||---|||
- MP3
- MP4
- M4A
- AAC
- FLAC
- OGG
- WAV
- AIFF
- APE
- DFF
- DSF
- OPUS
- SF2
- SFZ
- MPC
- MPP
- MP+
- TTA
|||---|||
